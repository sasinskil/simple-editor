# My simple editor

Come on, see live: [Live here](https://sasinskil.github.io/simple-editor)

## What to do to run the site locally ?

`npm install -g gulp-cli`

`npm install`

`gulp`

To ***publish*** with the ghelp ofithub pages use `npm run deploy`

### Technologies used

- HTML/HTML5,
- CSS/SCSS,
- JavaScript,
- Gulp
